Comp1950 Website- A Redesign

By Something From A hat

Project Description

The project creates a proto-site and documents to support the redesign of the COMP1950 web site.

The COMP 1950 owner (Alistair Calder) requires that the website be redesigned to provide better user-experience to students using the website. The site must comply to the requirements specified by the owner.

The principal aim of the project is for students to apply the various web-desgin and development techniques learnt during the course and also to work through the project using the project methodology formulated during class discussions.
